宝宅智能家居 是什么？

宝宅智能家居是一套简单的、开源的智能家居开发平台。主要面向对智能家居充满浓厚兴趣的朋友们，宝宅的目的是降低开门槛，基于宝宅，您不用投入太多的学习成本，不需要关心太多的技术细节，而是把更多的精力放到智能家居功能本身的开发上，轻松实现自己所想的功能。

宝宅智能家居开发流程

硬件开发

使用市面上最便宜的ESP8266 WIFI芯片，基于Arduino for ESP8266 开发环境，像开发arduino一样简单来开发设备程序。

网关开发

基于树莓派，网关程序一键安装，WEB版在线集成开发环境，采用javascript语言开发，上手容易，通过在网关上编写控制代码非常方便的实现硬件访问和联动
